    # 1. css设置子元素的宽度为父元素的一半

    当padding属性设置为百分值的时候，是以父元素的宽度为基准进行计算的，且只有设置padding-top和padding-bottom有效，padding-left和padding-right无效。css的padding和margin设置为百分比时，是相对于父元素的宽度来进行计算的。

    .child {
    width: 50%;
    padding-top: 50%;
    <!-- padding-bottom: 50%; -->
}
